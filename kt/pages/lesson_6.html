Литералы относятся к фиксированным значениям, которые программа не может изменять при её выполнении.
Они могут быть любого из основных типов данных, таких как целочисленная константа, плавающая константа, символьная константа или строковый литерал.

Kotlin поддерживает следующие литералы для интегральных значений:
- Decimals: 123 (Longs are tagged by a capital L: 123L")
- Hexadecimals: 0x0F
- Binaries: 0b00001011

Литералы октальных чисел (восьмеричные числа) не поддерживаются в Kotlin.

Вы можете использовать символы подчеркивания, чтобы сделать числовые константы более удобочитаемыми:

fun main(args: Array<String>) {
  val billion = 1_000_000_000;
  val hexNum = 0xAA_0C_BA_FF;
  val bytes = 0b11011011_11100010;

  // все будет выведено в 10-ой базе
  println(billion);
  println(hexNum);
  println(bytes);
}


Output:
1000000000
2852961023
56290

Поскольку числа с плавающей запятой по умолчанию интерпретируются как Double, Floats помечены тегами f или F.

fun main(args: Array<String>) {
  val thisIsFloat = 3.14f;
  val againFloat = 2.78F;
  val doubleIsDefault = 20.134;

  println(thisIsFloat);
  println(againFloat);
  println(doubleIsDefault);
}


Output:
3.14
2.78
20.134
